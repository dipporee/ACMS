       ex)

       c. '사업단, 1차계통 설계자, 2차계통 설계자' 선택한 경우, 총 23자로 '사업단 외 2건'

			if(instNm != '' && instNm.length > 20){

			}

    함 - '참조기관'이 2개 이상일 경우 글자수와 무관 : ‘OO 외 몇건’ 으로 표기
       ex) 
       a. '기타 설계사' 인 경우, '기타 설계사'
       b. '기타 설계사, 2차계통 구매자' 선택한 경우, '기타설계사 외 1건'
       c. 'T/D 제작사, 주설비 시공사, 부대설비 시공사' 선택한 경우, 'T/D 제작사 외 2건'

			if(instNm != ''){
